Сборка заперета от Nine_Teen (версия 1.0)

!!! Убедитесь что сборка никак не модифицирована кем либо из вне 

!!! Известно что ютубер peekbot распространяет зараженые сборки запрета ничего не скачаивайте с его телеграм канала

!!! Все компаненты запрета были взяты у bol-van. https://github.com/bol-van/zapret можете скачать по ссылке если не уверены что эта сборка чистая

!!! Если архив под паролем скорее всего это вирус и сборку следует удалить

!!! Самый простой способ проверить это вписать в поиск проводника "*.exe". Если найдется файл который весит больше 20мб скорее всего это вирус и сборку следует удалить

Передупреждение не только к этому архиву а ко всем впринцепе Если же вы уверен в содержимом архива то можете поьзоватся


Для обычного пользования:
1. Убедится что никаких обходов нет на пк.
2. Разархивировать содержимое.
3. Запустить general.bat или general_alt.bat и не закрывать консоль (свернуть можно)
после этого Ютуб и Дискорд должны быть разблокированы 

Для прекращения работы достаточно закрыть окно консоли и запустить stop.bat.


Если вдруг стандартные стратегии не подошли нужно подобрать новые с помощью blockcheck.

Для запуска blockcheck: 
1. Скачать java по этой ссылке https://download.oracle.com/java/17/archive/jdk-17.0.10_windows-x64_bin.exe (если есть недоверие то можете найти ее самостоятельно важно что бы версия была 17.0.10) 
 1.1 Что бы убедится что версия нужная достаточно открыть командную строку и вписать java -version
     Если все утановлено коректно то вывод должен быть такой:
	java version "17.0.10" 2024-01-16 LTS
	Java(TM) SE Runtime Environment (build 17.0.10+11-LTS-240)
	Java HotSpot(TM) 64-Bit Server VM (build 17.0.10+11-LTS-240, mixed mode, sharing)
2. Убедится что никаких обходов нет на пк. Если ранее запрет был запущен то запустите stop.bat что бы убедится что запрет выключен.
3. Запускаем Blockcheck.bat. 
 3.1. Проверка автоматическая в среднем занимает 10 минут. !!! Обязательно дождатся ее окончания.
 3.2. Больше одного раза проверку запускать нет смысла.
4. Как проверка закончится программа создаст новые general_[].bat файлы которые можно использовать так же как и general.bat или general_alt.bat.
